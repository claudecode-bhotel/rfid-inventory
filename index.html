DataWedge設定ガイド

Zebraデバイスで以下のDataWedgeプロファイルを設定してください:

  1. DataWedgeアプリを開く
  2. 新しいプロファイルを作成（名前: RFID在庫管理）
  3. 関連付けアプリ → 使用するブラウザを選択
    Chrome: com.android.chrome
    Enterprise Browser: com.symbol.enterprisebrowser
  4. Input → RFID Input → 有効化（Enabled）
  5. Output → Keystroke Output → 有効化（Enabled）
  6. キーストローク間遅延（Inter character delay） → 50ms
    ※ RFID入力には50ms必須
  7. Key event options → Send Characters as Events → チェックを入れる
    ※ ブラウザにキーイベントとして送信するために必須
  8. データ末尾アクション（Action key character） → Enter（改行）
トラブルシューティング
  • 「スキャン開始」をタップ後、入力フィールドに緑色の枠が表示されることを確認
  • RFIDトリガーを引いた後、入力フィールドに文字が表示されるか確認
  • 文字が表示されない場合 → DataWedgeプロファイルの「関連付けアプリ」がブラウザと一致しているか確認
  • Profile0（デフォルト）が干渉する場合 → Profile0を無効化
  • 123RFID Mobileで読み取れる場合はハードウェアは正常です
